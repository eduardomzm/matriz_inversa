Matriz Inversa
Transforma la matriz original en una matriz identidad donde 1 en diagonal y 0 en el resto mediante
operaciones de fila, aplicando las mismas operaciones matemáticas que incluyen sumas, restas y 
divisiones de filas para transformar ese cuadro en su versión inversa. La matriz debe ser cuadrada. 
Genera números decimalesincluso si la entrada son enteros. Por eso se modifico la clase Files.java
para agregar un método para manejar la precisión y no perder datos al redondear.
